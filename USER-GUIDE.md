# StarGaze 使用手册

欢迎使用 StarGaze 泰国艺人照片展示系统。本文档将指导您如何使用本系统的各项页面功能。

## 🌐 首页功能 (User Interface)

首页是用户交互的核心区域，集成了展示、搜索与快捷操作。

### 1. 氛围感背景
- **动态气泡**: 页面背景会自动生成随主题色变化的半透明气泡，营造轻盈的视觉效果。
- **主题切换**: 支持深色/浅色模式切换，适配您的视觉偏好。

### 2. 当月寿星 (Birthday Stars)
- **自动展示**: 首页顶部会自动滚动展示本月生日的艺人卡片。
- **快捷查看**: 点击艺人头像可直接进入其个人照片墙。

### 3. 照片墙与搜索
- **无限滚动**: 照片列表支持瀑布流展示，向下滚动自动加载更多。
- **智能搜索**: 顶部搜索栏支持按艺人姓名、作品、院校等关键词实时过滤。
- **多维度筛选**: 点击筛选按钮，可按年龄、身高、学历、出生月份等精细化查找。

### 4. 侧边快捷导航
- **悬浮球**: 页面右侧设有快捷导航球。
- **功能按钮**:
    - **回到顶部**: 一键返回页面最上方。
    - **主题切换**: 快速切换深色/浅色模式。

---

## 🔐 管理后台 (Admin Panel)

点击页面底部的“管理”链接并登录后即可进入后台。

### 1. 仪表盘
- 查看全站统计数据，包括总照片数、艺人数、院校数及本月新增照片。

### 2. 照片管理
- **批量上传**: 支持拖拽上传多张照片，并可一次性关联艺人和添加标签。
- **批量操作**: 在管理列表中，您可以勾选多张照片进行批量更新关联或批量删除。
- **设置首图**: 为艺人指定某张照片作为其个人主页和列表的展示封面。

### 3. 艺人资料管理
- 编辑艺人的详细档案（包括姓名、泰文名、身高体重、院校专业、代表作等）。
- 管理艺人标签，方便前台搜索。

### 4. 系统维护
- **数据导出/导入**: 支持将数据库内容导出为 JSON，或从旧版数据导入。
- **快照备份 (Snapshots)**: 
    - 一键创建数据库快照。
    - 随时从快照列表还原历史状态。
    - 下载 SQL 备份文件到本地保存。
- **系统日志**: 实时记录管理操作日志，确保 system 安全可追溯。

---

## 📱 移动端适配
- 系统采用响应式设计，所有功能在手机端均有良好的操作体验。
- 照片墙将自动切换为单/双列布局，方便触屏滑动。
